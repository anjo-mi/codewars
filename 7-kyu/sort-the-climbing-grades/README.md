<h2><a href=https://www.codewars.com/kata/58a08e622e7fb654a300000e/train/javascript target="_blank">Sort the climbing grades</a></h2><h3>7 kyu</h3><p>My 5th kata, and 1st in a planned series of rock climbing themed katas.</p><p>In rock climbing (<a href="https://en.wikipedia.org/wiki/Bouldering" data-turbolinks="false" target="_blank">bouldering</a> specifically), the V/Vermin (USA) climbing grades start at <code>'VB'</code> (the easiest grade), and then go <code>'V0'</code>, <code>'V0+'</code>, <code>'V1'</code>, <code>'V2'</code>, <code>'V3'</code>, <code>'V4'</code>, <code>'V5'</code> etc. up to <code>'V17'</code> (the hardest grade). </p><p>You will be given a <code>list</code> of climbing grades and you have to write a function to <code>return</code> a <code>list</code> of the grades sorted easiest to hardest.</p><p>If the input is an empty <code>list</code>, <code>return</code> an empty <code>list</code>; otherwise the input will always be a valid <code>list</code> of one or more grades.</p><p>Please do vote, rank, and provide any feedback on the kata.</p>
