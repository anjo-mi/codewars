<h2><a href=https://www.codewars.com/kata/584d2c19766c2b2f6a00004f/train/javascript target="_blank">Boolean logic from scratch</a></h2><h3>7 kyu</h3><h2 id="task">Task</h2><p>You need to implement two functions, <code>xor</code> and <code>or</code>, that replicate the behaviour of their respective operators:</p><ul><li><code>xor</code> = Takes 2 values and returns <code>true</code> if, and only if, one of them is truthy.</li><li><code>or</code> = Takes 2 values and returns <code>true</code> if either one of them is truthy.</li></ul><p>When doing so, <strong>you cannot use the or operator: <code>||</code></strong>.</p><h1 id="input">Input</h1><ul><li>Not all input will be booleans - there will be truthy and falsey values [the latter including also empty strings and empty arrays]</li><li>There will always be 2 values provided</li></ul><h2 id="examples">Examples</h2><ul><li><code>xor(true, true)</code> should return <code>false</code></li><li><code>xor(false, true)</code> should return <code>true</code></li><li><code>or(true, false)</code> should return <code>true</code></li><li><code>or(false, false)</code> should return <code>false</code></li></ul>
