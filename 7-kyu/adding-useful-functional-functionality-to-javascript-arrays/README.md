<h2><a href=https://www.codewars.com/kata/52195c9bb576caf14200007f/train/javascript target="_blank">Adding useful functional functionality to JavaScript arrays  </a></h2><h3>7 kyu</h3><p>The JavaScript standard now includes functional additions to array like <code>map</code>, <code>filter</code> and <code>reduce</code>, but sadly is missing the convenience functions <code>range</code> and <code>sum</code>.</p><p>Implement a version of <code>range</code> and <code>sum</code> (which you can then copy and use in your future kata to make them smaller).</p><p><code>Array.range(start, count)</code> should return an array containing <code>count</code> numbers from <code>start</code> to <code>start + count</code> ( exclusive )</p><p>Example: <code>Array.range(0, 3)</code> returns <code>[0, 1, 2]</code></p><p><code>array.sum()</code> should return the sum of all numbers in <code>array</code></p><p>Example: <code>[0, 1, 2].sum()</code> returns <code>3</code><br>Example: <code>Array.range(-1,4).sum()</code> should return <code>2</code></p><p>While not forbidden try to write both function without using a for loop</p>
