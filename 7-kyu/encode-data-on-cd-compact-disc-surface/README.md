<h2><a href=https://www.codewars.com/kata/643a47fadad36407bf3e97ea/train/typescript target="_blank">Encode data on CD (Compact Disc) surface</a></h2><h3>7 kyu</h3><p>"Pits" and "lands" are physical features on the surface of a CD that represent binary data. Pits are small depressions or grooves on the surface of the CD, while lands are flat areas between two adjacent pits.</p><p>The pits and lands themselves do not directly represent the zeros and ones of binary data. Instead, Non-return-to-zero, inverted encoding is used: a change from pit to land or land to pit indicates a one, while no change indicates a zero.</p><p>In this Kata, you should implement a function, that takes integer in range [0..255] (8 bits), and returns combination of pits and lands that encode the number.Result should have format of string: <code>PLLPL...</code> where <code>P</code> represents pit and <code>L</code> represents land. Combination should always start with pit.Numbers should be encoded in little-endian, which means you start from least significant bit.</p><h3 id="example">Example</h3><p>Input: <code>5</code></p><p>Binary representation (8 bits): <code>00000101</code></p><p>Output: <code>PLLPPPPPP</code></p><p>Explanation:</p><ol><li>Starts from P as per description</li><li>Changes to L because first bit is <code>1</code></li><li>Stays L because second bit is <code>0</code></li><li>Changes to P because third bit is <code>1</code></li><li>Stays P because fourth bit is <code>0</code></li><li>Stays P till the end because all subsequent bits are <code>0</code></li></ol>
