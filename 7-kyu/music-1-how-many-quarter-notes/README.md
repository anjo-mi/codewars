<h2><a href=https://www.codewars.com/kata/69c2f04a1294ffc95c526d9e/train/typescript target="_blank">Music 1: How Many Quarter Notes?</a></h2><h3>7 kyu</h3><p>Given a time signature (as a string), you must find the amount of quarter notes that can be fit into one measure, which will be returned as an unsigned integer. If a non-integer number of quarter notes can be fit into a measure, the value should be floored. <strong>DO NOT RETURN A NON INTEGER.</strong></p><p><strong>The Given Time Signature:</strong></p><ul><li>Will not be malformed</li><li>Will have a top number below 4096</li><li>Will have a bottom number below 256</li></ul><p>All valid time signatures must have a bottom number that is a power of 2. If the given signature is invalid, return <code>None</code> in Python, <code>null</code> in Javascript and TypeScript, or <code>-1</code> in C.</p><p>If less then one quarter note can be fit in a measure, return <code>0</code>;</p><p>Examples:</p><p><code>"3/4"</code> → <code>3</code></p><p><code>"7/8"</code> → <code>3</code></p><p><code>"11/8"</code> → <code>5</code></p><p><code>"10/7"</code> → <code>None</code>/<code>null</code>/<code>-1</code></p><p><code>"3/16"</code> → <code>0</code></p>
