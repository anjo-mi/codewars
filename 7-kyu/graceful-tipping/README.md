<h2><a href=https://www.codewars.com/kata/5eb27d81077a7400171c6820/train/typescript target="_blank">Graceful Tipping</a></h2><h3>7 kyu</h3><p>Adding tip to a restaurant bill in a graceful way can be tricky, thats why you need make a function for it.</p><p>The function will receive the restaurant bill (always a positive number) as an argument. You need to 1) <strong>add at least 15%</strong> in tip, 2) round that number up to an <em>elegant</em> value and 3) return it.</p><p>What is an <em>elegant</em> number? It depends on the magnitude of the number to be rounded. Numbers below 10 should simply be rounded to whole numbers. Numbers 10 and above should be rounded like this:</p><p>10 - 99.99... ---&gt; Round to number divisible by 5</p><p>100 - 999.99... ---&gt; Round to number divisible by 50</p><p>1000 - 9999.99... ---&gt; Round to number divisible by 500</p><p>And so on...</p><p>Good luck!</p><h2 id="examples">Examples</h2><pre><code> 1  --&gt;    2 7  --&gt;    912  --&gt;   1586  --&gt;  100</code></pre>
