<h2><a href=https://www.codewars.com/kata/551e51155ed5ab41450006e1/train/javascript target="_blank">Sum Times Tables</a></h2><h3>7 kyu</h3><p>Write a function which sums the result of the sums of the elements specified in <code>table</code> multiplied by all the numbers in between <code>min</code> and <code>max</code> including themselves.</p><p>For example, for <code>table=[2,5], min=1, max=3</code> the result should be the same as</p><pre><code>2*1 + 2*2 + 2*3 +5*1 + 5*2 + 5*3</code></pre><p>i.e. the table of two from 1 to 3 plus the table of five from 1 to 3</p><p>All the numbers are integers but you must take in account:</p><ul><li><code>table</code> could be empty.</li><li><code>min</code> could be negative.</li><li><code>max</code> could be really big.</li><li>It is guaranteed that the input <code>min</code> &lt;= <code>max</code></li></ul>
