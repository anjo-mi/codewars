<h2><a href=https://www.codewars.com/kata/5c857bab20089d4fe65a7f5b/train/typescript target="_blank">Remove Empty Items of Array</a></h2><h3>7 kyu</h3><p>In this Kata, you will learn how to remove all empty items in an Array.</p><p>In JavaScript, empty items in arrays can be declared by <code>[1, 2,,,3, 4]</code> (Multiple commas without a value in that index)</p><p>The values that are not given a value are empty items, and usually add up with others to form <code>&lt;# empty items&gt;</code>; In the example, <code>&lt;2 empty items&gt;</code></p><p><code>[1, 2, &lt;2 empty items&gt;, 3, 4]</code> should be cleared such that it should be <code>[1, 2, 3, 4]</code></p><p><strong>Example:</strong></p><p>Before: <code>[1, 2, 3, &lt;5 empty items&gt;, 4, &lt;1 empty item&gt;, null, undefined];</code></p><p>After: <code>[1, 2, 3, 4, null, undefined];</code></p>
