<h2><a href=https://www.codewars.com/kata/5753b987aeb792508d0010e2/train/javascript target="_blank">Every nth array element. (Basic)</a></h2><h3>7 kyu</h3><p>Create a function (a method in Ruby, an extension method in C#) <code>every</code> which returns every nth element of an array.</p><h3 id="usage">Usage</h3><ul><li><p>With one argument, <code>every(array)</code> returns every element of the <code>array</code>.  </p></li><li><p>With two arguments, <code>every(array, interval)</code> it returns every <code>interval</code>th element.  </p></li><li><p>With three arguments, <code>every(array, interval, start_index)</code> returns every <code>interval</code>th element starting at index <code>start_index</code>.</p></li></ul><h3 id="examples">Examples</h3><pre><code>every([0,1,2,3,4])     -&gt; [0,1,2,3,4]every([0,1,2,3,4],1)   -&gt; [0,1,2,3,4]every([0,1,2,3,4],2)   -&gt; [0,2,4]every([0,1,2,3,4],3)   -&gt; [0,3]every([0,1,2,3,4],3,1) -&gt; [1,4]</code></pre><h3 id="notes">Notes</h3><p>Test cases:  </p><ul><li><code>interval</code> will always be a positive integer (but may be larger than the size of the array).  </li><li><code>start_index</code> will always be within the bounds of the array.</li></ul><p>Once you have completed this kata, try the <strong>advanced version</strong> (<a href="http://www.codewars.com/kata/every-nth-array-element-advanced" data-turbolinks="false" target="_blank">http://www.codewars.com/kata/every-nth-array-element-advanced</a>) which allows negative intervals and unbounded start_indexes</p>
