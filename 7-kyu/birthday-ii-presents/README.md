<h2><a href=https://www.codewars.com/kata/5805f0663f1f9c49be00011f/train/typescript target="_blank">Birthday II - Presents</a></h2><h3>7 kyu</h3><p>Your colleagues have been good enough(?) to buy you a birthday gift. Even though it is your birthday and not theirs, they have decided to play pass the parcel with it so that everyone has an even chance of winning. There are multiple presents, and you will receive one, but not all are nice... One even explodes and covers you in soil... strange office. To make up for this one present is a dog! Happy days! (do not buy dogs as presents, and if you do, never wrap them).</p><p>Depending on the number of passes in the game (y), and the present you unwrap (x), return as follows:</p><p>x == goodpresent --&gt; return x with num of passes added to each charCode (turn to charCode, add y to each, turn back)<br><br>x == crap || x == empty --&gt; return string sorted alphabetically<br><br>x == bang --&gt; return string turned to char codes, each code reduced by number of passes and summed to a single figure<br><br>x == badpresent --&gt; return 'Take this back!'<br><br>x == dog, return 'pass out from excitement y times' (where y is the value given for y).</p>
