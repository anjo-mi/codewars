<h2><a href=https://www.codewars.com/kata/5a1c28f9c9fc0ef2e900013b/train/javascript target="_blank">Return pyramids</a></h2><h3>7 kyu</h3><p>The task is very simple: you must return pyramids. Given a number <code>n</code>,  you build a pyramid with <code>n</code> floors</p><p>For example , given a <code>n=4</code> you must to print this pyramid:</p><pre><code>   /\  /  \ /    \/______\ </code></pre><p>Other example, given a <code>n=6</code> you must to print this pyramid:</p><pre><code>     /\    /  \   /    \  /      \ /        \/__________\</code></pre><p>Another example, given a <code>n=10</code>, you must to print this pyramid:</p><pre><code>         /\        /  \       /    \      /      \     /        \    /          \   /            \  /              \ /                \/__________________\</code></pre><p>Note: a line feed character is needed at the end of the string.</p>
