<h2><a href=https://www.codewars.com/kata/5803753aab6c2099e600000e/train/java target="_blank">Age in days</a></h2><h3>7 kyu</h3><p>Did you ever want to know how many days old are you? Complete the function which returns your age in days. The birthday is given in the following order: <code>year, month, day</code>.</p><p>For example if today is 30 November 2015 then </p><pre><code>ageInDays(2015, 11, 1) returns "You are 29 days old"</code></pre><p>The  birthday is expected to be in the past.</p><p>Suggestions on how to improve the kata are welcome!</p>
