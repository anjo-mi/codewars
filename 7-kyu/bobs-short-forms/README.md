<h2><a href=https://www.codewars.com/kata/570cbe88f616a8f4f50011ac/train/javascript target="_blank">Bob's Short Forms</a></h2><h3>7 kyu</h3><p>Bob is a theoretical coder - he doesn't write code, but comes up with theories, formulas and algorithm ideas. You are his secretary, and he has tasked you with writing the code for his newest project - a method for making the short form of a word. Write a function <code>shortForm</code>(C# <code>ShortForm</code>, Python <code>short_form</code>) that takes a string and returns it converted into short form using the rule: <em>Remove all vowels, except for those that are the first or last letter. </em>Do not count 'y' as a vowel, and ignore case. Also note, the string given will not have any spaces; only one word, and only Roman letters. <br>Example:</p><pre><code>shortForm("assault");short_form("assault")ShortForm("assault");// should return "asslt"</code></pre><p><br> <br>Also, FYI: I got all the words with no vowels from <br><a href="https://en.wikipedia.org/wiki/English_words_without_vowels" data-turbolinks="false" target="_blank">https://en.wikipedia.org/wiki/English_words_without_vowels</a></p>
