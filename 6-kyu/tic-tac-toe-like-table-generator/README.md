<h2><a href=https://www.codewars.com/kata/5b817c2a0ce070ace8002be0/train/javascript target="_blank">Tic-Tac-Toe-like table Generator</a></h2><h3>6 kyu</h3><p>Do you have in mind the good old TicTacToe?</p><p>Assuming that you get all the data in one array, you put a space around each value, <code>|</code> as a columns separator and multiple <code>-</code> as rows separator, with something like <code>["O", "X", " ", " ", "X", " ", "X", "O", " "]</code> you should be returning this structure (inclusive of new lines):</p><pre><code> O | X |   -----------   | X |   ----------- X | O |   </code></pre><p>Now, to spice up things a bit, we are going to expand our board well beyond a trivial <code>3</code> x <code>3</code> square and we will accept rectangles of big sizes, still all as a long linear array.</p><p>For example, for <code>"O", "X", " ", " ", "X", " ", "X", "O", " ", "O"]</code> (same as above, just one extra <code>"O"</code>) and knowing that the length of each row is <code>5</code>, you will be returning</p><pre><code> O | X |   |   | X -------------------   | X | O |   | O </code></pre><p>And worry not about missing elements, as the array/list/vector length is always going to be a multiple of the width.</p>
