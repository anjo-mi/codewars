<h2><a href=https://www.codewars.com/kata/5934d648d95386bc8200010b/train/javascript target="_blank">Ka Ka Ka cypher - words only vol 1</a></h2><h3>6 kyu</h3><h1 id="introduction">Introduction</h1><p>Ka ka ka cypher is a cypher used by small children in some country. When a girl wants to pass something to the other girls and there are some boys nearby, she can use Ka cypher. So only the other girls are able to understand her. <br>She speaks using KA, ie.: <br><code>ka thi ka s ka bo ka y ka i ka s ka u ka gly</code> what simply means <code>this boy is ugly</code>. <br></p><h1 id="task">Task</h1><p>Write a function that accepts a string word and returns encoded message using ka cypher. <br><br>Our rules:<br></p><ul><li>The encoded word should start from <code>ka</code>.</li><li>The <code>ka</code> goes after vowel (a,e,i,o,u)</li><li>When there is multiple vowels together, the <code>ka</code> goes only after the last <code>vowel</code></li><li>When the word is finished by a vowel, do not add the <code>ka</code> after</li></ul><h1 id="inputoutput">Input/Output</h1><p>The <code>word</code> string consists of only lowercase and uppercase characters. There is only 1 word to convert - no white spaces.</p><h1 id="example">Example</h1><pre><code>"a" =&gt; "kaa""ka" =&gt; "kaka""aa" =&gt; "kaaa"  "Abbaa" =&gt; "kaAkabbaa""maintenance" =&gt; "kamaikantekanakance""Woodie" =&gt; "kaWookadie""Incomprehensibilities" =&gt; "kaIkancokamprekahekansikabikalikatiekas"</code></pre><h1 id="remark">Remark</h1><p>Ka cypher's country residents, please don't hate me for simplifying the way how we divide the words into "syllables" in the Kata. I don't want to make it too hard for other nations ;-P </p>
