<h2><a href=https://www.codewars.com/kata/596e91b48c92ceff0c00001f/train/typescript target="_blank">Words to Hex</a></h2><h3>6 kyu</h3><p>Gary likes pictures but he also likes words and reading. He has had a desire for a long time to see what words and books would look like if they could be seen as images.</p><p>For this task you are required to take a continuous string that can consist of any combination of words or characters and then convert the words that make up this string into hexadecimal values that could then be read as colour values.</p><p>A word is defined as a sequence of ASCII characters between two white space characters or the first or last word of a sequence of words. </p><p>Each word will represent a hexadecimal value by taking the first three letters of each word and find the ASCII character code for each character. This will then give you one hexadecimal value that represents the colours red, green or blue. You will then combine these values into one readable RGB hexadecimal value, e.g. <code>#ffffff</code>.</p><p>If your word consists of less than 3 letters, you should use the hexidecimal value <code>00</code>, ie <code>"It"</code> would return a value <code>#497400</code>.</p><p>Your answer should be an array of hexadecimal values that correspond to each word that made up your original string.</p><h2 id="example">Example</h2><p>The following string would be given:</p><p><code>"Hello, my name is Gary and I like cheese."</code></p><p>This string would return the following array:</p><pre><code>['#48656c', '#6d7900', '#6e616d','#697300','#476172','#616e64','#490000','#6c696b','#636865']</code></pre>
