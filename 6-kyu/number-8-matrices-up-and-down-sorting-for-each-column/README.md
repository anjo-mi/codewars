<h2><a href=https://www.codewars.com/kata/590b8d5cee471472f40000aa/train/typescript target="_blank">#8 Matrices: Up and Down Sorting For Each Column</a></h2><h3>6 kyu</h3><p>You are given a matrix <code>M</code>, of positive and negative integers. It should be sorted in an up and down column way, starting always with the lowest element placed at the top left position finishing with the highest depending on <code>n</code> value: at the bottom right position if the number of columns,<code>n</code>,  is odd, or placed at the top right, if <code>n</code> is even.</p><p>E.g.:</p><pre><code>M = [[-20, -4, -1],     [  1,  4,  7],      [  8, 10, 12]]     M_ = [[-20, 7, 8],      [-4, 4, 10],      [-1, 1, 12]]</code></pre><p>In order to be more understandable we show the directions of the sorting for the new matrix with arrows:</p><p><a href="http://imgur.com/owCZAeI" data-turbolinks="false" target="_blank"><img title="source: imgur.com" src="http://i.imgur.com/owCZAeI.jpg"></a></p><p>Create the function <code>up_down_col_sort()</code> that receives a matrix as an argument and may do this kind of sorting.</p><p>Your function will be tested with square or rectangular matrices of <code>m</code> rows and <code>n</code> columnsFeatures of the random tests:</p><pre><code>Number of tests = 12010 &lt;= m &lt;= 20010 &lt;= n &lt;= 200-1000 &lt;= M[i,j] &lt;= 1000</code></pre><p>The kata is available at Python 2, Typescript, Javascript and Ruby at the moment.Translations into another languages will be released soon.</p>
