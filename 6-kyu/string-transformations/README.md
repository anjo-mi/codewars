<h2><a href=https://www.codewars.com/kata/6927680a91f3d91da9d2b16c/train/javascript target="_blank">String transformations</a></h2><h3>6 kyu</h3><p>You are given a string and a transformation pattern (an array of operations). Your task is to apply the operations in order to the string and return the final result.</p><p>Each operation in the pattern is one of:</p><p>"U" → convert the string to uppercase</p><p>"L" → convert the string to lowercase</p><p>"R" → reverse the string</p><p>"D" → duplicate <strong>each</strong> character (e.g., "abc" → "aabbcc", "qqbbcc" → "qqqqbbbbcccc")</p><pre><code>stringTransformation("abcd", ["U","R"]) // "DCBA"// "abcd" → "ABCD" → "DCBA"stringTransformation("abc", ["D","L"])  // "aabbcc"// "abc" → "aabbcc" → "aabbcc" (lowercase already applied)stringTransformation("AbCd", ["L","R"]) // "dcba"// "AbCd" → "abcd" → "dcba"</code></pre><p><em><strong>Note</strong></em>:</p><p>Input string length is <strong>1 &lt;= n &lt;= 10^5</strong>, length of transformations is <strong>0 &lt;= n &lt;= 10^5</strong>, in all tests is guaranteed that the string will be <strong>not longer than 10^8</strong>.</p>
