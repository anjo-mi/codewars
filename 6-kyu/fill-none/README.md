<h2><a href=https://www.codewars.com/kata/5edaa69f5fcd510020199489/train/typescript target="_blank">Fill None</a></h2><h3>6 kyu</h3><h2 id="task">Task</h2><p>Write a function that accepts a list that <em>can</em> contain missing data, and an integer representing the method on how to fill the missing data if there is any.  Missing data is represented as <code>None</code>. The list will only contain integers and <code>None</code> values.</p><p>Note that depending on the language you attempt this kata with, <code>None</code> corresponds to:</p><ul><li><code>None</code> (Python)</li><li><code>undefined</code> (Javascript and TypeScript)</li><li><code>Nothing</code> (Haskell)</li><li><code>null</code> (C#)</li></ul><p>The fill method rules are outlined below:</p><pre><code>Fill method:   -1: backwards    0: nearest         1: forwards</code></pre><h2 id="example">Example</h2><pre><code>arr = [None, 1, None, None, None, 2, None]fill(arr, -1) == [1, 1, 2, 2, 2, 2, None]  # None replaced by closest int on the rightfill(arr,  0) == [1, 1, 1, 1, 2, 2, 2]     # None replaced by closest int. If equidistant, choose the smallest intfill(arr,  1) == [None, 1, 1, 1, 1, 2, 2]  # None replaced by closest int on the left</code></pre><h3 id="notes">Notes</h3><ul><li><code>[]</code> should return <code>[]</code></li><li><code>[None]</code> should return <code>[None]</code></li><li>Arrays will only contain integers and <code>None</code> values</li></ul>
