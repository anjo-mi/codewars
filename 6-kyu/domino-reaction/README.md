<h2><a href=https://www.codewars.com/kata/584cfac5bd160694640000ae/train/typescript target="_blank">Domino Reaction</a></h2><h3>6 kyu</h3><p>You're given a string of dominos. For each slot, there are 3 options:</p><ul><li><p>"|" represents a standing domino</p></li><li><p>"/" represents a knocked over domino</p></li><li><p>" " represents a space where there is no domino</p></li></ul><p>For example: </p><pre><code>"||| ||||//| |/"</code></pre><p>Tipping a domino will cause the next domino to its right to fall over as well, but if a domino is already tipped over, or there is a domino missing, the reaction will stop.<br>What you must do is find the resulting string if the first domino is pushed over. So in out example above, the result would be:</p><pre><code>"/// ||||//| |/"</code></pre><p>since the reaction would stop as soon as it gets to a space.</p>
