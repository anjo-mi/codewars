<h2><a href=https://www.codewars.com/kata/62aa9a45732360005cb69c26/train/typescript target="_blank">Reverse a Tuple (Type)</a></h2><h3>6 kyu</h3><p>Reverse a tuple... at the type level.</p><p>In other words, if you have</p><pre><code>type Foo = [string, number, boolean]</code></pre><p>then <code>Reverse&lt;Foo&gt;</code> should be the type <code>[boolean, number, string]</code></p><p><code>Reverse</code> should also reject non-tuple inputs; in other words, <code>Reverse&lt;1&gt;</code> (for example) should be a type error.</p><p>Since this is type-level programming, success is simply making the program compile.</p>
