<h2><a href=https://www.codewars.com/kata/534161c506654f1559000483/train/typescript target="_blank">Simplify Compound Fractions</a></h2><h3>6 kyu</h3><p>Write a function that will simplify a compound fraction. As input, it should accept three arguments as integers: integer, numerator, and denominator. It should return an array with three items, all integers: <code>[integer, numerator, denominator]</code>. If there should be no integer, or no fraction resulting from simplification, just return zeros in their place.</p><p>There is a convenience function you can use called <code>toString(integer, numerator, denominator)</code>. It also takes three integer values, and returns a string of the compound fraction for logging, etc.</p><p>For example:</p><pre><code class="language-javascript"><span class="cm-comment">// 4 3/2 (four and three halves)</span><span class="cm-variable">simplify</span>(<span class="cm-number">4</span>, <span class="cm-number">3</span>, <span class="cm-number">2</span>); <span class="cm-comment">// == [5, 1, 2]</span><span class="cm-comment">// becomes 5 1/2 (five and one half)</span><span class="cm-comment">// 15/12 (fifteen twelfths)</span><span class="cm-variable">simplify</span>(<span class="cm-number">0</span>, <span class="cm-number">15</span>, <span class="cm-number">12</span>); <span class="cm-comment">// == [1, 1, 4]</span><span class="cm-comment">// becomes 1 1/4 (one and one quarter)</span></code></pre><blockquote><p><strong>Note:</strong> Beware of 0 denominators and numerators.</p></blockquote>
